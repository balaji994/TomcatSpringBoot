# README #

### Requirements ###

* Java 8
* Maven 3.3.x

### Building the artifact ###

```
mvn clean package
```

### Running the application from command line ###

```
mvn spring-boot:run
```

### Available URLs

Sending requests to:
```
http://localhost:8880/demo

http://localhost:8882/demo
```
should result in successful responses.
